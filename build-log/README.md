# Build Log

*How Tariqa was built. What worked, what didn't, what we learned.*

Entries are added as things are built — not before.

---

## Format

Each entry is a markdown file named by date and topic:
`YYYY-MM-topic.md`

Contents:
- What we built
- Why
- How (enough detail to reproduce)
- What surprised us
- What we'd do differently
